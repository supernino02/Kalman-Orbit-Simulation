# Directory Organization

This document explains the organization of the project directory in detail.

- data/: Contains all the CSV datasets used and generated by the project, including the simulated states (orbiting.csv) and the noisy measurements (orbiting_noisy.csv).
- docs/: Contains documentation files, such as this docs.md file. It is the designated location for any auxiliary documentation.
- 
otebooks/: Contains the core Jupyter notebooks. 
    - REAL_WORLD_SIMULATION.ipynb: Generates the underlying simulation of the spaceship and adds structural noise.
    - FINAL_PPRESENTATION.ipynb: The final presentation notebook, applying the Kalman filter to trace and correct the state estimations.
- ideos/: Stores the output .mp4 video files generated by the simulations, which visually demonstrate the spaceship movements and Kalman predictions.
- equirements.txt: A plain text file defining the Python package dependencies required to run the notebooks (e.g., numpy, plotly, opencv-python).
- README.md: The main entry point explaining the project, containing the conceptual overview and setup instructions.
